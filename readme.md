# 📚 个人学习笔记管理系统 (Personal Note System)

这是一个基于 **Spring Boot 3** 和 **Vue 3** 构建的高性能、现代化的个人知识管理平台。系统集成了富文本编辑、智能标签提取、多维度检索、数据可视化分析以及夜间模式，旨在为用户提供极致的笔记记录与复盘体验。

## ✨ 核心功能

### 1. 笔记核心管理
* **富文本编辑**：集成 WangEditor v5，支持图文混排、代码块等。
* **智能自动保存**：基于 3 秒防抖逻辑（Debounce），在用户停止输入后自动备份，确保数据永不丢失。
* **生命周期管理**：支持“正常-回收站-彻底删除”三段式逻辑，防止误删。
* **星标收藏**：快速标记重要笔记，支持一键收藏/取消。

### 2. 深度检索与组织
* **多维度筛选**：支持按关键词（标题/内容）、分类、标签、星标状态及日期范围进行服务端组合检索。
* **最近访问**：首页自动记录并展示前 8 篇最近阅读或编辑的笔记，实现快速回溯。
* **智能标签系统**：后端算法自动提取笔记高频关键词生成标签，支持点击标签一键筛选。

### 3. 数据备份与分析
* **多格式导出**：支持单篇笔记导出为 Markdown (.md) 或纯文本 (.txt)。
* **分类批量备份**：支持按分类一键打包导出 ZIP 压缩包，文件名完美适配中文。
* **可视化看板**：利用 ECharts 展示分类占比图、创作趋势图、高频搜索词云及 24 小时活跃时段分布。

### 4. 用户系统与交互
* **账号安全**：基于 JJWT 实现无状态身份验证，密码采用 BCrypt 高强度哈希存储。
* **找回密码**：通过“用户名 + 注册邮箱”验证实现密码重置。
* **个人中心**：支持自定义头像、昵称及个性签名。
* **夜间模式**：全局适配亮色/暗色主题，支持持久化存储用户偏好。

## 🛠️ 技术栈

* **后端**: Java 17, Spring Boot 3.2.4, Spring Data JPA, MySQL 8, JJWT, jBCrypt, Lombok.
* **前端**: Vue 3 (Composition API), Vite, Element Plus, ECharts, Axios, Vue Router.

## 🚀 快速开始

### 前置要求
* JDK 17+
* Node.js 18+
* MySQL 8.0+

### 1. 数据库配置
运行项目根目录下的 `note_system.sql` 文件初始化表结构。

### 2. 后端启动
1.  修改 `backend/src/main/resources/application.yml` 中的数据库连接信息。
2.  在 `backend` 目录下运行：
    ```bash
    mvn spring-boot:run
    ```

### 3. 前端启动
1.  进入 `frontend` 目录。
2.  安装依赖：`npm install`
3.  启动开发服务器：
    ```bash
    npm run dev
    ```
4.  访问地址：`http://localhost:5173`（默认测试账号需先注册）。

## 📂 项目结构
* `backend/`: 基于 Spring Boot 的 RESTful API。
* `frontend/`: 基于 Vue 3 的单页面应用 (SPA)。
* `uploads/`: 物理映射目录，用于存储用户上传的头像。
